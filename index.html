NodeJS i18n

Minimalistic internationalization using gettext style for NodeJS.

Features

  • Gettext style
  • Easy of use
  • Create .po via CLI
  • Select lang per user

How to Use

Example

const { Resources, Translation } = require('nodejs-i18n')

const r = new Resources()
r.load('pt', 'pt.po')

var user = new Translation('pt')
var world = user._`world` // Mundo
user._`Hello ${world}` // Ol√° mundo

Options :: Resources({})

  • lang - String

    Lang id.
    Default value: en

Options :: Resources({}).load(lang, file)

Load translation

  • lang - String

    Lang id.

  • file - String

    File path

Options :: Resources({}).remove()

Remove translation.

  • lang - String

    Lang id.

Options :: Translation(lang)

Select translation.

  • lang - String

    Lang id.

Options :: Translation(lang)._(string)

String translation.

  • String - String

CLI

Create .po file with nodejs-i18n. Example: add in package.json

"scripts": {
    "i18n": "find . | grep .js | sort | nodejs-i18n > en.po"
}

and run $ npm run i18n

Installation

This is a Node.js module available through the npm registry. It can be installed using the npm or yarn command line tools.

npm install nodejs-i18n --save

Dependencies

  • gettext-parser: Parse and compile gettext po and mo files to/from json, nothing more, nothing less
  • tempy: Get a random temporary file or directory path

Dev Dependencies

  • mocha: simple, flexible, fun test framework
